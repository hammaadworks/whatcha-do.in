### üöÄ **Your Journey Awaits: Stop Hesitating. Start Dominating.**

The moment is now. The tool is here. What are you waiting for?

1.  **Claim your destiny:**
    ```bash
    git clone https://github.com/hammaadworks/whatcha-do.in.git
    cd whatcha-doin
    ```
2.  **Forge your tools:**
    ```bash
    pnpm install
    ```
3.  **Prepare your command center:**
    Create a `.env.local` file based on `.env.local.example`. You'll need:
    *   `NEXT_PUBLIC_SUPABASE_URL`
    *   `NEXT_PUBLIC_SUPABASE_ANON_KEY`
    *   `NEXT_PUBLIC_DEV_USER`

4.  **Ignite the engine. Witness the dawn of a new you:**
    ```bash
    pnpm dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser. The transformation begins the moment you do.
